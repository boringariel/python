# matplotlib

-   ### matplotlib 불러오기

    > 파이썬을 실행한 뒤, import 구문을 이용한다.<br>
    > 기본적인 그래프를 그릴 수 있는 pyplot을 불러온다.

    ```{.Python}
    import matplotlib.pyplot as plt
    ```

    > 위 소스코드에서, x와 y축에 대응되는 데이터를 입력하면 간단하게 그래프를 작성할 수 있다.<br>
    > 이는 plot()구문과 show()구문을 이용한다.

    ```{.python}
    plt.plot(x,y)
    plt.show()
    ```
